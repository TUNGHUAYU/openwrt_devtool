## Usage

### New package ( prpl plugin )
``` bash
    $ bash devtool.sh add ${OPENWRT_DIR} ${PKG_NAME}
```

### Modify existed package 
``` bash 
    $ bash devtool.sh modify ${OPENWRT_PKG_DIR}
```
